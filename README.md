# routemap-express-mw

Express middleware for writing elegant apis.

## Motivation

## Usage

Install `routemap-express-mw`.

```
npm install routemap-express-mw --save
```

In your server code, such as app.js:

```
var app = require('express')();

// mw to write elegant apis
app.use(routeMap());
```
